# Master Thesis

This repository contains the code and Quarto sources for a thesis on high-dimensional instrumental variable quantile regression (IVQR) under weak instruments.

The planned estimators are:

- Full-control IVQR
- Post-selection IVQR
- DML-IVQR

The code goal is to run a Monte Carlo simulation study comparing these estimators across relevant data-generating processes.

Current stage: project structure and DGP implementation.
